## 詳細要件

### ボード

| # | 機能 | 詳細 | 受け入れ基準 |
|---|------|------|------------|
| B-1 | 初期データ表示 | アプリ起動時に「To Do / In Progress / Done」の3列とサンプルカードを表示する | ・初回アクセス時に3列が左から順に表示される<br>・「To Do」列に「サンプルタスク」カードが1件表示される |
| B-2 | データ永続化 | ボードのすべての変更はバックエンド API 経由で即時 PostgreSQL へ保存される。ページリロード時は API から最新状態を取得して復元する | ・カード追加後にページをリロードしても追加したカードが表示される<br>・ブラウザを閉じて再度開いても状態が維持される |

### 列（カラム）管理

| # | 機能 | 詳細 | 受け入れ基準 |
|---|------|------|------------|
| C-1 | 列の追加 | 「+ 列を追加」ボタンから任意のタイトルで列を追加できる | ・ボタンクリックでタイトル入力フォームが表示される<br>・タイトルを入力し「追加」ボタンで新しい列が右端に追加される |
| C-2 | 列の追加 (Enter) | タイトル入力中に Enter キーで追加を確定できる | ・入力フォームで Enter キーを押すと列が追加される |
| C-3 | 列名の変更 | 列タイトルをクリックするとインライン編集できる | ・タイトルをクリックするとテキスト入力に変わる |
| C-4 | 列名の変更 (確定) | フォーカスを外す（blur）または Enter キーで変更を確定する | ・Enter キーまたはフォーカスアウトで変更が保存される<br>・空文字の場合は変更が保存されない |
| C-5 | 列の削除 | 列右上の「×」ボタンで列を削除する。列内のカードも同時に削除される | ・「×」ボタンクリックで列が消える<br>・削除した列内のカードも同時に削除される |
| C-6 | カード枚数表示 | 列ヘッダーに含まれるカード枚数をバッジで表示する | ・カードを追加・削除するたびに枚数が即座に更新される |

### カード管理

| # | 機能 | 詳細 | 受け入れ基準 |
|---|------|------|------------|
| K-1 | カードの追加 | 各列の「+ カードを追加」ボタンからカードを追加できる | ・ボタンクリックでタイトル・説明・優先度の入力フォームが表示される |
| K-2 | カードの追加 (入力) | タイトル（必須）、説明（任意）、優先度（HIGH / MEDIUM / LOW、デフォルト MEDIUM）を入力してカードを作成する | ・タイトルが空の場合は追加されない<br>・タイトル入力後「追加」ボタンでカードが列の末尾に追加される<br>・説明は空でも追加できる<br>・優先度はドロップダウンで選択する（未選択時は MEDIUM）|
| K-3 | カードの編集 | カード上の「編集」ボタンからタイトル・説明・優先度をインライン編集できる | ・「編集」クリックでタイトル・説明・優先度がテキスト入力／ドロップダウンに変わる<br>・「保存」クリックで変更が反映される<br>・「キャンセル」クリックで変更前の状態に戻る |
| K-4 | カードの削除 | カード上の「削除」ボタンでカードを削除する | ・「削除」クリックでカードが即座に消える |
| K-5 | 説明の表示 | 説明が設定されている場合のみタイトル下に表示する | ・説明が空の場合はタイトルのみ表示される<br>・説明がある場合はタイトル下に小さいフォントで表示される |
| K-6 | 優先度の表示 | 各カードの右上に優先度バッジを常時表示する | ・HIGH: 赤（`#ef4444`）・MEDIUM: 橙（`#f59e0b`）・LOW: 緑（`#22c55e`）の色付きバッジで表示される<br>・バッジには「高」「中」「低」のラベルテキストも表示される |

### ドラッグ&ドロップ

| # | 機能 | 詳細 | 受け入れ基準 |
|---|------|------|------------|
| D-1 | 列内の並び替え | カードを同じ列内でドラッグして順序を変更できる | ・ドラッグして離した位置にカードが移動する |
| D-2 | 列間の移動 | カードを別の列へドラッグ&ドロップして移動できる | ・別の列にドロップするとそのカードが移動する |
| D-3 | ドロップゾーンのハイライト | ドラッグ中のカードが列上にある時、列の背景色を変化させてフィードバックを与える | ・ドラッグ中に列上にカーソルがあると列背景が薄い青色に変わる |
| D-4 | ドラッグ中のカードのハイライト | ドラッグ中のカードの背景色を変化させる | ・ドラッグ中のカードは薄い青紫色になる |
| D-5 | 優先度順ソート | 列ヘッダーの「優先度順」ボタンで、その列のカードを HIGH → MEDIUM → LOW の順に並び替えて保存する | ・ボタンクリックで列内カードが優先度降順（HIGH 先頭）に並び替えられる<br>・並び替え結果はバックエンド API へ即時保存される |

### 検索

| # | 機能 | 詳細 | 受け入れ基準 |
|---|------|------|------------|
| S-1 | カード全文検索 | ヘッダーの検索バーにキーワードを入力し、タイトルまたは説明に一致するカードを一覧表示する | ・Enter キーまたは「検索」ボタンで検索が実行される<br>・一致したカードが所属する列ごとにグループ化して表示される<br>・検索結果画面中は通常のボード操作は行われない<br>・「クリア」ボタンで通常のボード表示に戻る<br>・一致件数がヘッダー下に表示される<br>・一致するカードがない場合はその旨のメッセージが表示される |

## ユースケース・操作フロー

### UC-1: タスクを追加する

```
ユーザー                          システム
   |                                 |
   |-- 「+ カードを追加」クリック ----->|
   |                                 |-- 入力フォームを列下部に展開
   |<--------------------------------|
   |-- タイトルを入力                  |
   |-- 説明を入力（任意）              |
   |-- 優先度を選択（任意）            |
   |-- 「追加」クリック --------------->|
   |                                 |-- バリデーション（タイトル必須）
   |                                 |-- POST /api/cards でバックエンドへ保存
   |                                 |-- カードを列末尾に追加（楽観的更新）
   |<-- カードが列に表示される ---------|
```

### UC-2: タスクを別の列に移動する（ドラッグ&ドロップ）

```
ユーザー                          システム
   |                                 |
   |-- カードをドラッグ開始 ----------->|
   |                                 |-- カードをハイライト（薄い青紫）
   |-- 別の列にホバー --------------->|
   |                                 |-- 列をハイライト（薄い青）
   |-- ドロップ ---------------------->|
   |                                 |-- カードを移動先列の指定位置に挿入（楽観的更新）
   |                                 |-- PUT /api/board/reorder でバックエンドへ保存
   |<-- ボードが更新される ------------|
```

### UC-3: 列を追加する

```
ユーザー                          システム
   |                                 |
   |-- 「+ 列を追加」クリック --------->|
   |                                 |-- タイトル入力フォームを展開
   |<--------------------------------|
   |-- タイトルを入力                  |
   |-- 「追加」クリックまたは Enter ---->|
   |                                 |-- 列をボード右端に追加（楽観的更新）
   |                                 |-- POST /api/columns でバックエンドへ保存
   |<-- 列が追加される ---------------|
```

### UC-4: タスクを編集する

```
ユーザー                          システム
   |                                 |
   |-- カードの「編集」クリック -------->|
   |                                 |-- タイトル・説明・優先度を編集フォームに切り替え
   |<--------------------------------|
   |-- タイトル・説明・優先度を変更     |
   |-- 「保存」クリック --------------->|
   |                                 |-- カードを更新（楽観的更新）
   |                                 |-- PATCH /api/cards/:id でバックエンドへ保存
   |<-- 変更が反映される -------------|
```

### UC-5: 列名を変更する

```
ユーザー                          システム
   |                                 |
   |-- 列タイトルをクリック ----------->|
   |                                 |-- タイトルをテキスト入力に切り替え
   |<--------------------------------|
   |-- タイトルを変更                  |
   |-- Enter キー押下またはフォーカスアウト->|
   |                                 |-- 列タイトルを更新（楽観的更新）
   |                                 |-- PATCH /api/columns/:id でバックエンドへ保存
   |<-- タイトルが変更される ----------|
```

### UC-6: タスクを検索する

```
ユーザー                          システム
   |                                 |
   |-- 検索バーにキーワードを入力 ------>|
   |-- Enter またはボタンクリック ----->|
   |                                 |-- GET /api/board/search?q=キーワード
   |                                 |-- タイトルまたは説明に一致するカードを列ごとに集計
   |<-- 検索結果画面を表示 ------------|
   |                                 |
   |-- 「クリア」ボタンクリック -------->|
   |<-- 通常ボード表示に戻る ----------|
```

### UC-7: 優先度順に並び替える

```
ユーザー                          システム
   |                                 |
   |-- 列ヘッダーの「優先度順」クリック->|
   |                                 |-- 列内カードを HIGH → MEDIUM → LOW 順に並び替え（楽観的更新）
   |                                 |-- PUT /api/board/reorder でバックエンドへ保存
   |<-- 列内のカードが優先度順に並ぶ ---|
```
