# タスクボード アプリ 要件定義書

**プロジェクト名:** タスクボード (🐾 タスクボード)  
**作成日:** 2026-05-23  
**バージョン:** 1.0.0

---

## 1. プロジェクト概要

### 1.1 目的

カンバン方式のタスク管理ボードを提供し、タスクの状態を視覚的に整理・管理できるようにする。

### 1.2 対象ユーザー

個人でタスクを管理したいユーザー。

### 1.3 利用環境

- Webブラウザ（デスクトップ）
- データはブラウザのローカルストレージに保存（サーバー不要）

## 3. 機能要件

### 3.1 ボード

| # | 機能 | 詳細 |
|---|------|------|
| B-1 | 初期データ表示 | アプリ起動時に「To Do / In Progress / Done」の3列とサンプルカードを表示する |
| B-2 | データ永続化 | ボードの状態をlocalStorageに自動保存し、リロード後も復元する |

### 3.2 列（カラム）管理

| # | 機能 | 詳細 |
|---|------|------|
| C-1 | 列の追加 | 「+ 列を追加」ボタンから任意のタイトルで列を追加できる |
| C-2 | 列の追加 (Enter) | タイトル入力中にEnterキーで追加を確定できる |
| C-3 | 列名の変更 | 列タイトルをクリックするとインライン編集できる |
| C-4 | 列名の変更 (確定) | フォーカスを外す（blur）またはEnterキーで変更を確定する |
| C-5 | 列の削除 | 列右上の「×」ボタンで列を削除する。列内のカードも同時に削除される |
| C-6 | カード枚数表示 | 列ヘッダーに含まれるカード枚数をバッジで表示する |

### 3.3 カード管理

| # | 機能 | 詳細 |
|---|------|------|
| K-1 | カードの追加 | 各列の「+ カードを追加」ボタンからカードを追加できる |
| K-2 | カードの追加 (入力) | タイトル（必須）と説明（任意）を入力してカードを作成する |
| K-3 | カードの編集 | カード上の「編集」ボタンからタイトルと説明をインライン編集できる |
| K-4 | カードの削除 | カード上の「削除」ボタンでカードを削除する |
| K-5 | 説明の表示 | 説明が設定されている場合のみタイトル下に表示する |

### 3.4 ドラッグ&ドロップ

| # | 機能 | 詳細 |
|---|------|------|
| D-1 | 列内の並び替え | カードを同じ列内でドラッグして順序を変更できる |
| D-2 | 列間の移動 | カードを別の列へドラッグ&ドロップして移動できる |
| D-3 | ドロップゾーンのハイライト | ドラッグ中のカードが列上にある時、列の背景色を変化させてフィードバックを与える |
| D-4 | ドラッグ中のカードのハイライト | ドラッグ中のカードの背景色を変化させる |

## 4. 非機能要件

### 4.1 UI / UX

| # | 要件 |
|---|------|
| U-1 | 温かみのあるベージュ系の背景色 (`#fef9f0`) を使用する |
| U-2 | 動物の絵文字をグリッド状に薄く敷いた装飾的な背景を表示する |
| U-3 | ヘッダーはスクロール時も固定表示し、すりガラス風のエフェクトを適用する |
| U-4 | 列は横スクロールで閲覧できる |
| U-5 | 列の最大高さをビューポート高に合わせ、カードが多い場合は列内でスクロールする |

### 4.2 データ

| # | 要件 |
|---|------|
| Da-1 | すべてのデータ変更（追加・編集・削除・並び替え）は即座にlocalStorageへ反映する |
| Da-2 | localStorageのデータが破損している場合はデフォルトデータにフォールバックする |

## 5. データ構造

```typescript
// カード
interface Card {
  id: string        // ユニークID
  title: string     // タイトル（必須）
  description: string // 説明（任意、空文字可）
}

// 列（カラム）
interface Column {
  id: string        // ユニークID
  title: string     // 列のタイトル
  cardIds: string[] // カードIDの順序付きリスト
}

// ボード全体
interface BoardData {
  columns: Record<string, Column>  // 列のマップ
  cards: Record<string, Card>      // カードのマップ
  columnOrder: string[]            // 列の表示順序
}
```
