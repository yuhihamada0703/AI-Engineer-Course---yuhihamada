以下の手順でバックエンドとフロントエンドのサーバーを起動してください。

## 必須: ポート競合の確認と解消

1. まず `lsof -i:8080 -i:5173` を実行してポートの使用状況を確認する
2. 競合プロセスが存在する場合は `kill <PID>` で停止する
3. ポートが解放されたことを `lsof -i:8080 -i:5173` で確認する

**重要:** 別のポートで起動することは絶対に禁止。必ず指定ポートで起動する。

## バックエンド起動 (Port: 8080)

```bash
cd backend && ./gradlew bootRun
```

起動後、`curl -sf http://localhost:8080/api/board` でレスポンスが返ることを確認する。

## フロントエンド起動 (Port: 5173)

```bash
npm run dev
```

起動後、`curl -sf http://localhost:5173` でHTMLが返ることを確認する。

## 固定ポート一覧

| サービス | ポート |
|---------|--------|
| フロントエンド (Vite) | 5173 |
| バックエンド (Spring Boot) | 8080 |
| データベース (PostgreSQL) | 5432 |
